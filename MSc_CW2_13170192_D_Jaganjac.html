Scaling the variables gives each variable equal importance in the hierarchical clustering performed. In my opinion, we should scale the variables before the inter-observation dissimilarities are computed. It is important to do this in order for each variable to be given equal weighting therefore giving a more accurate result. For example, for the USArrests dataset, many more assaults are committed than murders of rapes per 100,000 residents. Without scaling, more importance would then be given to assault than the other variables as assaults occur more often. By scaling, the variables are given equal importance and so murders and rapes are seen as equal to assault. This can be seen visually in part.c as the first dendrogram is more balanced compared to the second which is skewed to the left.
